# 🚀 TechStore
Aplicação web desenvolvida em **React** com foco em performance, escalabilidade e experiência do usuário. O projeto simula uma loja virtual moderna, aplicando boas práticas de desenvolvimento front-end, integração com APIs e conceitos de computação em nuvem.

---

## 🎓 Projeto Acadêmico
Este projeto foi desenvolvido como parte da disciplina **Desenvolvimento de Software em Nuvem**, da **Universidade de Fortaleza (UNIFOR)**.

A proposta da disciplina envolve a aplicação prática de conceitos como:
- Arquitetura em nuvem ☁️  
- Consumo e integração com APIs REST 🔗  
- Escalabilidade e organização de sistemas 📈  
- Práticas modernas de DevOps 🚀  

---

## 📌 Sobre o Projeto
O **TechStore** é uma aplicação front-end que simula um e-commerce de produtos tecnológicos, com foco em:

- Interface moderna
- Componentização com React  
- Organização escalável de código  
- Integração com serviços externos (APIs)  
- Experiência do usuário aprimorada com animações

## ☁️ Arquitetura e DevOps
O projeto foi estruturado considerando boas práticas de **arquitetura em nuvem** e **DevOps**, incluindo:

- Separação entre front-end e back-end em repositórios distintos  
- Integração com serviços externos (Firebase / Supabase)  
- Versionamento com Git e GitHub  
- Estrutura preparada para CI/CD  
- Deploy em nuvem (GCP)  

### 🔄 Estratégia de Deploy
- Front-end preparado para deploy em ambientes como:
  - Google Cloud Platform (GCP)
  - Vercel / Netlify  

- Back-end utilizando serviços gerenciados (BaaS):
  - Firebase  
  - Supabase  

- Pipeline de CI/CD (conceitual):
  - Build automatizado  
  - Testes (futuro)  
  - Deploy contínuo
